## Dataset Description

The dataset contains social media posts from the PersonalityCafe forum, where users have self-reported their MBTI personality type. Each row represents one user and contains:

- **`type`**: The user's 4-letter MBTI personality type (e.g., `INTJ`, `ENFP`)
- **`posts`**: The user's last 50 posts, separated by `|||`

There are 16 possible personality types, combining:
- **I/E** — Introversion / Extraversion  
- **N/S** — Intuition / Sensing  
- **T/F** — Thinking / Feeling  
- **J/P** — Judging / Perceiving

## How to Obtain the Dataset

> **Manual download required** — the dataset is too large to include directly in the repository.

### Step-by-step instructions:

1. Go to: [https://www.kaggle.com/datasets/datasnaek/mbti-type](https://www.kaggle.com/datasets/datasnaek/mbti-type)
2. Log in to your Kaggle account (create one for free if needed)
3. Click the **Download** button (top right)
4. Extract the downloaded zip file
5. Locate `mbti_1.csv`
6. Place it in this folder: `data/raw/mbti_1.csv`
