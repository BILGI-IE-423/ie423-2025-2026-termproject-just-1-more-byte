# Outputs

This folder contains all output files generated by the project scripts.

## Folder Structure

```
outputs/
├── figures/    ← Generated plots (.png)
└── tables/     ← Generated tables (.csv)
```

## Contents

### `figures/`
| File | Description |
|------|-------------|
| `fig1_type_distribution.png` | MBTI type distribution (bar chart) |
| `fig2_dimension_balance.png` | I/E, N/S, T/F, J/P balance across 4 dimensions |
| `fig3_wordcount_IE.png` | Post length comparison by I/E type |
| `fig4_wordcount_boxplots.png` | Word count distribution across all MBTI types |
| `fig5_vocabulary_richness.png` | Vocabulary richness by type (unique word ratio) |

### `tables/`
| File | Description |
|------|-------------|
| `type_distribution.csv` | Row count and percentage for each MBTI type |
| `missing_value_summary.csv` | Missing value summary |
| `linguistic_summary_by_type.csv` | Linguistic statistics by type |

## Note
These files are not pushed to GitHub (excluded via `.gitignore`).  
You can generate them locally by running:

```bash
python scripts/03_basic_eda.py
```
